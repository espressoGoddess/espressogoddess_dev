Bingo

Bingo is a full-stack bingo application with dynamic game board generation and user authentication. It is built using a normalized database schema to ensure data integrity and minimize duplication and authentication with Google OAuth via Next-Auth for secure user authentication.

  • TypeScript
  • Next.js
  • Supabase
  • Next-auth
  • Tailwind

Beam

Beam is a self-care app designed for users to process and manage their emotions. Users select a feeling, reflect on it, and note insights to prevent or recreate the emotion. The app then suggests relevant activities. Entries are stored in the Feelings Archive for ongoing review and editing.

  • PostgreSQL
  • Express.js
  • React
  • Node
  • Cypress
